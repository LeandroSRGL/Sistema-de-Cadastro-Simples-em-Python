# 📋 Cadastro de Clientes

Sistema de cadastro de clientes em Python com persistência em JSON.

## Funcionalidades

- Cadastrar clientes (nome, CPF, telefone, e-mail e endereço)
- Buscar cliente por CPF
- Alterar dados de um cliente
- Apagar um cliente
- Listar todos os clientes cadastrados

## Como funciona

Os dados são salvos em um arquivo `clientes.json` e mantidos entre execuções do programa.
